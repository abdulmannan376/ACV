### Semantic Segmentation
Implementation of SegNet, UNet, and DeepLabV3plus for Semantic Segmentation of nuclei in digital histology images using Keras. [**Dataset**](https://monuseg.grand-challenge.org/Data/) contains multi organ tissue images with the ground truth segmentation masks for nucli.
### Requirements
* Keras 2.x
* Python 3.x
* Tensorflow 1.x
